Build 0.1.zip 사용법

0. 실행 방법
설치 후 압축을 해제하여 응용프로그램 Rook Bishop Knight를 실행

1. 맵 에딧 방법
-우측하단 녹화버튼(빨간 원)을 클릭하여 녹화 시작.
-단축키를 눌러 설치하고자 하는 아이템을 선택

	0: 빈칸
	1: 일반 칸
	2: 플레이어 룩
	3: 플레이어 비숍
	4: 플레이어 나이트
	P: 적 폰(현재 모델은 비숍)
	R: 적 룩
	B: 적 비숍
	N: 적 나이트
	Q: 적 퀸
	K: 적 킹

-선택한 아이템은 Moves 판넬 위에 표시됨
-블록을 클릭해서 아이템을 배치 
-마우스를 누른 상태로 끌어서 여러개의 아이템을 한번에 배치할 수 있습니다.
-맵 아이템 배치가 완료된 상태이면 다시 녹화 버튼을 클릭(이때 맵의 정보가 로컬 세이브파일로 저장됩니다.)
-맵을 플레이하여 클리어(모든 적 유닛을 플레이어 유닛으로 밟아 제거되어야 하며, 적 유닛이 1개 이상이어야 클리어가 가능합니다.)
-클리어 결과창에서 edit 버튼을 눌러 최단 클리어 턴수와 스테이지 이름을 각각 활성화된 입력창을 클릭해 입력.
-입력이 완료되면 다시 edit 버튼을 누르고 save 버튼을 눌러 저장.
-결과창의 버튼이 아닌 부분을 클릭하여 다음 스테이지 생성.

2. 스테이지 이동 방법
-녹화가 아닌 상태에서 사용 가능.(녹화중인 경우에는 녹화버튼을 눌러 플레이 화면으로 나와서 사용 가능)
-F1을 누르면 스테이지 이름 부분에 입력창이 활성화됨.
-입력창에 이동하고자 하는 스테이지 숫자를 입력 후 다시 F1을 누르면 해당 스테이지 로드

3. 세이브 파일 위치
C:\Users\user\AppData\LocalLow\RubyKnights\Rook Bishop Knight\mapdata.save
이 파일을 공유하여 덮어쓰면 다른 사람이 만든 맵을 플레이 할 수 있음. (자신의 데이터 파일은 백업해둘것!)

4. 발견된 버그
-여러개의 플레이어 말 배치 가능(실제로 사용 가능한건 책 읽는 순서대로 스캔했을 때 마지막에 있는 플레이어 말)
-블록이 아니라 블록 위에 놓인 아이템을 클릭할 경우 수정이 안됨
-룩과 비숍이 빈칸을 건너갈 수 있음
